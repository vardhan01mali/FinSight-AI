# FinSight AI - Financial Document Intelligence Platform

FinSight AI is a production-ready, lightweight, and completely free-to-run AI Financial Document Intelligence platform using RAG (Retrieval-Augmented Generation). It is optimized to run seamlessly on low-RAM laptops by utilizing a smart hybrid architecture: heavy LLM inference is delegated to Groq's high-speed cloud API, while document chunk embedding generation (`sentence-transformers`) and vector storage (`ChromaDB` with disk persistence) run locally.

## Features

- **Multi-Format Parsing:** Support for PDF, Word (DOCX), Excel (XLSX, XLS), CSV, TXT, JSON, XML, and Images (JPG, JPEG, PNG).
- **OCR Fallback:** Automatic OCR extraction using Tesseract OCR if direct text parsing of a document/image returns empty content.
- **Strict User Isolation:** Built-in JWT-based session security. Document metadata, metrics, chats, and vector embeddings are securely isolated using individual `user_id` query filters.
- **Interactive Financial Dashboard:** Synthesized key metrics (Revenue, Expenses, Profit, Assets, Liabilities, Cash Flow, Debt) rendered on visual charts via Recharts.
- **Document Variance Analysis:** Side-by-side comparison tables calculating absolute and percentage differences between any two filings.
- **AI Narrative Synthesis:** Smart summaries generated by LLM analysis describing comparisons or executive summaries.
- **ReportLab PDF & CSV Exports:** Clean exports of comparative narratives and metric tables.
- **Robust RAG Q&A:** Multi-page search query references displaying page numbers, file sources, and matching excerpts.
- **API Quality Controls:** Built-in endpoint rate limiting (15 queries/min for chats) and FastAPI Swagger documentation.

## 🛠️ Local Environment Setup

### Prerequisites

- **Python:** Python 3.10 or 3.11 installed.
- **Node.js:** Node.js (v18+ or v20+) and npm.
- **Tesseract OCR:** Required for scanned documents and image OCR fallback.
  - *Windows:* Download installer from UB Mannheim. Add Tesseract path (typically `C:\Program Files\Tesseract-OCR`) to your system environment `Path` variables.
  - *Linux:* `sudo apt install tesseract-ocr libtesseract-dev poppler-utils`

### 1. Backend Setup

1. Open a terminal in the `backend/` directory:
   ```bash
   cd backend
   ```
2. Create and activate a virtual environment:
   ```bash
   python -m venv venv
   # On Windows:
   .\venv\Scripts\activate
   # On Linux/macOS:
   source venv/bin/activate
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Create the `.env` configuration file inside the `backend` folder (based on `.env.example` template):
   ```ini
   GROQ_API_KEY=your_actual_groq_api_key
   AI_PROVIDER=groq
   DATABASE_URL=sqlite:///./finsight.db
   JWT_SECRET=supersecretjwtkeyforfinsightai1234567890
   CHROMA_PATH=./chroma_db
   ALLOWED_ORIGINS=http://localhost:5173
   MAX_FILE_SIZE=20971520
   ```
   *Note:* Replace `your_actual_groq_api_key` with your Groq API key from [Groq Console](https://console.groq.com).
5. Start the backend developer server:
   ```bash
   uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
   ```
   Open `http://localhost:8000/docs` to view the FastAPI Swagger API documentation.

### 2. Frontend Setup

1. Open a new terminal in the `frontend/` directory:
   ```bash
   cd frontend
   ```
2. Install Node packages:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
   Open `http://localhost:5173` in your browser to interact with the application.

## 🐳 Docker Deployment (Recommended)

To spin up the entire production-ready application stack with a persistent PostgreSQL database, FastAPI backend, and Nginx frontend, use Docker Compose:

1. Create a `.env` file at the project **root** directory (containing your Groq key):
   ```ini
   GROQ_API_KEY=your_actual_groq_api_key
   JWT_SECRET=use_a_custom_generated_long_hex_string
   ```
2. Start the Docker services:
   ```bash
   docker-compose up --build -d
   ```
3. Access the services:
   - **Frontend App:** `http://localhost` (served on standard port 80)
   - **Backend API:** `http://localhost:8000`

## 🔑 Groq Cloud API Key Setup

This application is built with a zero-cost API guarantee. It uses **Groq** for high-speed LLM inference:

1. Go to [Groq Console](https://console.groq.com) and create a free account.
2. Navigate to **API Keys** and generate a new API key.
3. Paste the key in your `.env` configuration file under the `GROQ_API_KEY` setting.
4. If the key is missing, the backend `/health` endpoint will report a `degraded` state and API requests trying to hit Groq will output:
   `"Groq API key missing. Add GROQ_API_KEY in .env file."`
